Coding region deletions

This page lists coding region deletions relative to the SARS-CoV-2 reference sequence have been detected in viral genome sequences from the outbreak.

Note that CoV-GLUE only detects deletions strictly within coding regions. A 382nt deletion which spans ORF 7b and ORF 8 has been confirmed in sequences from Singapore, but such deletions are not yet detected.
