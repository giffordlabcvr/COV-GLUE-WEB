Excluded sequences

CoV-GLUE excludes the following GISAID sequences from its analysis:
  • Any sequence of length less than 29000 nucleotides
  • Any sequence with a non-human host, e.g. bat, pangolin
  • Sequences from environmental samples
  • Any sequences marked with a warning flag, as having quality issues on GISAID
  • The following sequences thought to contain sequencing errors: EPI_ISL_406592, EPI_ISL_406595.
  • Any sequence which we detect has a frameshifting deletion or insertion relative to the SARS-COV-2 reference sequence.
