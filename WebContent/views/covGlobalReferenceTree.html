Global reference tree

CoV-GLUE dynamically computes a reference tree to represent the main lineages of the pandemic at the global level. Some sequences are excluded from the tree to keep it manageable. The reference tree is computed as follows:
  • Only sequences that pass the exclusion criteria are candidates for inclusion in the reference tree
  • The program cd-hit-est (see here) is applied to the coding regions of candidate sequences to generate clusters, using a 99.95% sequence identity threshold.
  • Each sequence that cd-hit-est selects as a cluster representative is included in the tree.
  • Additionally, non-representative sequences from each cluster are included in the tree to ensure that at least one sequence from each country represented in the cluster is included
  • The reference strain Wuhan-Hu-1 is always included in the tree.
  • The tree is computed using RAxML.
  • The rooting is based on an early split in the virus population, characterised by the L/S polymorphism at ORF 8 position 84.
